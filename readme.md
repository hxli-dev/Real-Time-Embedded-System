  (How to run visualizer: 
    1. Run the project in VS code. Only build+upload, don't start the monitor! 
    2. Open terminal
    3. cd :path to Real-Time-Embedded-System project's folder
    4. Run this line in terminal: python3 visualizer/visualizer.py)

- BLE Integration:
  include-ble_rtes.h
  
  (How to run BLE Integration:
    1. Install LightBlue or other BLE software on the phone
    2. Build+Upload the project in VS code. 
    3. Search your board name on LightBlue + Connect the board
    4. Subscribe the data, and you will see the BLE notifications)
